From Hackathon to Production

Free-Tier Launch

Supabase (500MB), Vercel hosting, OpenWeatherMap (1,000 calls/day), WAQI (unlimited). Supports 3 cities at 15-min polling with zero infrastructure cost.

Horizontal Scaling

FastAPI async workers scale horizontally. ML models served via stateless endpoints. PostgreSQL supports read replicas. Architecture designed for multi-region deployment.

IRDAI Regulatory Path

IRDAI Sandbox 2025 provides 6-12 month testing window. Precedent: Go Digit settled India's first parametric claim (Jan 2025). No regulatory blocker exists for gig-worker extension.

ML Feedback Loop

Week 1-4: Community-rated pricing. Week 5-12: Individual claim history incorporated. Month 4+: Full personalisation with weekly XGBoost refits on actual trigger-vs-predicted data.

SmartShift Loss Reduction

LSTM model predicts 48-hour disruption probability per shift block. Workers shift hours to green windows. Reduces claim payouts by ~15%: the insurer saves money, the worker earns more.

Guidewire-Native

PolicyCenter, ClaimCenter, BillingCenter map 1:1. Zero legacy integration for carriers. Trigger-to-payout runs within Guidewire's claims workflow with parametric logic layered on top.
